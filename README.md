# bt_api_coindcx

CoinDCX exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_coindcx
```

## Usage

```python
from bt_api_coindcx import CoinDCXExchangeDataSpot, CoinDCXRequestDataSpot
```

## Features

- REST API support
- HMAC-SHA256 authentication
- Symbol format: `BTCUSDT` (concatenated uppercase)
