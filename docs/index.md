# COINDCX Documentation

## English

Welcome to the COINDCX documentation for bt_api.

### Quick Start

```bash
pip install bt_api_coindcx
```

```python
from bt_api_coindcx import CoindcxApi
feed = CoindcxApi(api_key="your_key", secret="your_secret")
ticker = feed.get_ticker("BTCUSDT")
```
